## 科目三考试

-1.上车准备

-报告老师，学员某某某，准备完毕，请求考试，然后环车一周，报告老师，环车一周检视完毕，请求上车，开车门前左右摇头，确认安全才能上车

-2.上车后，关车门，系安全带，调座位，室内检车，确认档杆是否在空挡，手刹是否拉起，然后钥匙通一档电，调整后视镜和中间镜子，看前照灯是否关闭，灯杆往前抬两下，检查好后，报告老师，室内检查完毕，请求开始考试。

-3.听到请起步，先打开左转向灯，按喇叭，左右摇头观察，观察到前方有障碍物就不要挂挡，放手刹，如果没有障碍物就挂挡，放手刹，然后观察后视镜，确认安全后，起步，起步后，观察后视镜，确认安全后，打开右转向灯，按喇叭，然后观察后视镜，确认安全后，直行。车动了5秒之后关闭转向灯。
