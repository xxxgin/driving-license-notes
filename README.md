##科目三考试

###上车准备

-报告老师，学员某某某，准备完毕，请求考试，然后环车一周，报告老师，环车一周检视完毕，请求上车，开车门前左右摇头，确认安全才能上车

-上车后，关车门，系安全带，调座位，室内检车，确认档杆是否在空挡，手刹是否拉起，然后钥匙通一档电，调整后视镜和中间镜子，看前照灯是否关闭，灯杆往前抬两下，检查好后，报告老师，室内检查完毕，请求开始考试。

-听到请起步，先打开左转向灯，按喇叭，左右摇头观察，观察到前方有障碍物就不要挂挡，放手刹，如果没有障碍物就挂挡，放手刹，然后观察后视镜，确认安全后，起步，起步后，观察后视镜，确认安全后，打开右转向灯，按喇叭，然后观察后视镜，确认安全后，直行。车动了5秒之后关闭转向灯。

###行驶
-五号线和六号线，开到路口打右转向灯，车正以后踩油门加二档，出门口带刹车，出门以后在实线处停车，离合踩到底，轻踩刹车，左右摇头，回空挡，松离合

-确认安全以后，左右摇头，然后挂一档，起步，然后观察后视镜，确认安全后，打开右转向灯，转入最右侧车道，踩一脚油门，松开，离合踩到底，加二档，，而后听语音播报操作

###注意事项
-1.过斑马线，要注意红绿灯，绿灯时，斑马线前有横线，轻踩刹车，左右摇头，而后驶入，红灯时直接距离斑马线两三米时离合踩到底，轻踩刹车，车头将箭头盖住以后停车等待红灯，绿灯亮起后，离合踩到底，左右摇头，挂一档起步，踩一脚油门，松开，离合踩到底，加二档，而后听语音播报操作

-2.遇到公交车站，右手边的路口，学校标志，轻踩刹车，左右摇头

-3.百米加减档，二挡时踩油门到底，加速到28码，而后挂三挡，加速到38码，挂四挡，松开离合和刹车，而后保持5秒，五秒后，离合踩到底，踩刹车，挂二挡

-4.掉头，听见语音播报，打左转向灯，斑马线前轻踩刹车，左右摇头，驾驶过斑马线以后，左右摇头，观察左后视镜，安全后方向盘向左打死，驶入车道

-5.到考场门口，人行横道踩刹车，左右摇头，打右转向灯，驶入考场，六号线往左转，五号线右转

-6.靠边停车，提前把离合踩一半，听见语音播报打右转向灯，左右摇头，雨刮扫线，合适以后离合踩到底，踩刹车，停下以后，关灯光，回空挡，拉手刹，熄火，解开安全带，报告老师，操作完毕，请求下车，左右摇头观察，看左后视镜，开门下车，环车，报告老师，考试完毕，请求点评。
